## 技术细节

### 计分逻辑（最终版）
```javascript
if (a.v > 0) {
    STATE.s[dim[0]] += a.v;  // +2/+1 → 左侧维度加分
} else if (a.v < 0) {
    STATE.s[dim[1]] += Math.abs(a.v);  // -1/-2 → 右侧维度加分
}
// 0 → 都不加分（中立）
```

### 置信度算法
```javascript
置信度 = 50 + 分差贡献 (0-30) + 积极性贡献 (0-20) - 时间惩罚 (0/20)
分差贡献 = 各维度分差平均值 × 3
积极性贡献 = 总分 / 2
时间惩罚 = 平均每题<3 秒 扣 20 分
```
